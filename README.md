# Javascript Prosjekt for opptak ifra nettleser.
<h3> Dette er mitt praksis prosjekt som er gjort sammen med Tingtun. Jeg fikk som oppgave å lage ett system som gjør det mulig for brukere å kunne ta ett skjermopptak med lyden ifra mikrofonen direkte ifra nettleseren, uten å trenge noe eksterne hjelpemidler. Opptakene blir direkte lagret på Tingtun sin server inni en docker container uten at brukeren trenger å gjøre noe.

</h3>
