# Javascript Prosjekt for opptak ifra nettleser.
<h4> Dette er mitt praksis prosjekt som er gjort sammen med Tingtun. Jeg fikk som oppgave å lage ett system som gjør det mulig for brukere å kunne ta ett skjermopptak med lyden ifra mikrofonen direkte ifra nettleseren, uten å trenge noe eksterne hjelpemidler. Opptakene blir sendt og lagret direkte på Tingtun sin server uten at brukeren trenger å gjøre noe.</h4>  
<h4>
  Applikasjonen er skrevet i javascript, og filene sendes til en docker container og lagres direkte til filsystemet med logiske navn som blir bestemt i JS logikken.
</h4>
